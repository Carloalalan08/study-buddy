# Study Buddy: Smart Learning Dashboard

**Study Buddy** is a productivity-focused student web app that helps learners stay organized and motivated. It combines motivational quotes, task tracking, Pomodoro-style study sessions, and AI-powered topic explanations.

---

## Features

- To-do list with due dates and persistence
- Pomodoro-style study timer
- Daily motivational quote (via Quotable API)
- “Explain This” helper (via Twinword API)
- Productivity score tracker

---

## Tech Stack

- HTML5, CSS3, JavaScript (ES6+)
- APIs:
  - [Quotable](https://github.com/lukePeavey/quotable) – motivational quotes
  - [Twinword](https://www.twinword.com/api/) – definitions & explanations
  - [Mocky.io](https://designer.mocky.io/) – mock API for to-dos
